# Gmail Cleaner Project

## Description
This project aims at helping users clean up emails in their gmail account.

## Gmail Sender Count
The code in this file will count the number of emails sent by each sender in the user's gmail account. It outputs the results to a Google Sheet.
### Deployment
1. Copy the code in the file `gmail-sender-count.js`.
2. Go to [Google Apps Script](https://script.google.com/).
3. Create a new project.
4. Paste the code in the editor.
5. Run the main function `countGmailSender`.
6. Accept the permissions as required.
